Instructions on how to Compile the prototype

1. Tools needed
- Visual Studio Code
- Live Server extension

---

2. Download the Project
To get the project on your device:

1. Go to the repository and **download the ZIP file**.  
2. After it downloads, **extract** the ZIP.  
3. Move the extracted folder into your preferred location (you can put it inside a new folder if you want).  
4. Open VS Code, then go to **File → Open Folder…**  
5. Choose the extracted folder to load the project into VS Code.

---

3. Install Live Server
1. Click the **Extensions** icon on the left sidebar in VS Code.  
2. Search for **Live Server**.  
3. Install the one created by *Ritwick Dey*.  

---

## 4. Run the Project
1. Open the main HTML file (`tester.html`).  
2. Right-click the file and select **Open with Live Server**.  
   - Or just click the **Go Live** button at the bottom-right corner of VS Code.
